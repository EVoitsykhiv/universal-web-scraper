# Universal Web Scraper

Flexible Python CLI tool for extracting data from websites.

## Features

- Scrape any website
- Extract links, emails or page text
- Command line interface
- Export results to CSV

## Technologies

- Python
- Requests
- BeautifulSoup

## Installation

pip install -r requirements.txt

## Usage

Scrape links from a website:

python scraper.py https://example.com links

Scrape emails:

python scraper.py https://example.com emails
